### Getting Started 🐶

1. **시작하기**

   링크로 접속한 후 play 버튼을 클릭합니다.

2. **기록 달성하고 랭킹 등록하기 **

   주어진 시간 안에 닥스훈트를 모두 찾으면 미션 성공! 다음 레벨로 넘어갑니다.

   게임을 멈추거나 핫도그를 클릭하면 스코어, 플레이시간, 획득 점수를 토대로 랭크를 조회합니다.

   전체 플레이어 중 5위 이내에 들 경우 랭킹 등록이 가능합니다.
